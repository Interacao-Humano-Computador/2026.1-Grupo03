# Ferramentas

Abaixo estão listadas as ferramentas utilizadas no andamento do projeto.

- **GitHub:** Repositório do código e da documentação.
- **MkDocs:** Gerador do site estático de documentação.
- **Figma / Penpot:** Ferramenta(s) para prototipação e wireframes.
- **Teams / Discord / WhatsApp:** Ferramentas de comunicação da equipe.
