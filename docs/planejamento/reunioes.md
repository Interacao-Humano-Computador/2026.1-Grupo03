# Reuniões

## Introdução

Nesta página, encontra-se o registro oficial de todos os nossos encontros síncronos e reuniões de equipe. A documentação sistemática desses encontros visa garantir a transparência das decisões técnicas e gerenciais, além de servir como um histórico confiável para acompanhamento do progresso do projeto ao longo da disciplina e facilitar o resgate de pautas antigas.

## Modelo de Ata Utilizado

As atas de reunião do projeto seguem um modelo padronizado para garantir consistência, rastreabilidade e completude nas informações registradas. O modelo adotado pela equipe pode ser consultado e baixado no link abaixo:

[📄 Baixar Modelo de Ata (PDF)](../atas/SiglaProjeto_AtaReuniao_AAAAMMDD_XX.pdf){: target="_blank" }

O documento contempla os seguintes campos obrigatórios:

| Campo | Descrição |
|:---|:---|
| **Projeto / Disciplina** | Identificação do projeto e da disciplina vinculada |
| **Data** | Data de realização da reunião |
| **Horário de início e fim** | Intervalo de duração do encontro |
| **Local** | Plataforma ou local onde ocorreu (ex: Teams, Discord) |
| **Participantes** | Nome de todos os presentes |
| **Objetivo** | Finalidade principal da reunião |
| **Atividades Definidas** | Tarefas deliberadas, responsáveis e prazos |
| **Outros Assuntos** | Pontos adicionais discutidos |
| **Próxima Reunião** | Data e pauta prevista para o próximo encontro |
| **Aprovação** | Assinaturas de confirmação dos participantes |
