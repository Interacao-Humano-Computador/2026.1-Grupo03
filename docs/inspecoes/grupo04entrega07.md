# Inspeção da Entrega 07 - Grupo 04

## Introdução

Este documento contém o acompanhamento da verificação e do controle de qualidade de todas as entregas efetuadas pelo **Grupo 04** na **Etapa 7** do projeto. A modalidade de verificação adotada é a inspeção, com base na lista de verificação consolidada para o artefato: **Relato dos Resultados do Protótipo de Alta Fidelidade**.

A inspeção valida cada critério definido e apresentado na Lista de Verificação da Etapa 7, indicando a conformidade com a bibliografia da disciplina. Aplicamos o *color code* para facilitar a visualização: <span style="background-color: #2e7d32; color: white; padding: 4px 8px; border-radius: 4px; font-size: 12px; font-weight: bold;">Conforme</span> ou <span style="background-color: #d32f2f; color: white; padding: 4px 8px; border-radius: 4px; font-size: 12px; font-weight: bold;">Não Conforme</span>.
