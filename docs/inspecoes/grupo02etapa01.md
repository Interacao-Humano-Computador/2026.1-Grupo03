# Inspeção da Entrega 01 - Grupo 02

## Introdução

Este documento contém o acompanhamento da verificação e do controle de qualidade de todas as entregas efetuadas pelo **Grupo 02** na **Etapa 1** do projeto. A modalidade de verificação adotada é a inspeção, com base na lista de verificação consolidada para os artefatos: **Planejamento do Projeto, equipe, lista de sites avaliados, site selecionado para o projeto da disciplina, Ferramentas do projeto, Processo de Design e cronograma das atividades**.

A inspeção valida cada critério definido e apresentado na Lista de Verificação da Etapa 1, indicando a conformidade com a bibliografia da disciplina.
