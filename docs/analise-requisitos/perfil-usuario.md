# Perfil de Usuário

## Introdução

O Perfil de Usuário é uma descrição detalhada das características dos usuários cujos objetivos devem ser apoiados pelo sistema sendo projetado. Segundo Barbosa e Silva (2021), conhecer o perfil de usuário é fundamental para orientar as decisões de design, pois permite que a equipe compreenda quem são as pessoas que utilizarão o sistema, como elas se comportam, quais são suas necessidades, experiências e expectativas. A definição do perfil de usuário é uma das atividades iniciais do processo de design centrado no ser humano.
