# Ferramentas Utilizadas

## Introdução

Nesta página estão listadas as principais ferramentas empregadas ao longo do desenvolvimento do projeto de Interação Humano-Computador. A definição e o uso dessas ferramentas foram organizados por etapas para garantir a rastreabilidade e facilitar a compreensão do fluxo de trabalho da equipe.

---

> A associação de cada ferramenta às etapas foi consolidada a partir dos artefatos documentados no projeto (diagramas, protótipos, gravações e páginas de documentação).
